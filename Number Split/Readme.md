### Числовое разделение

Учитывая число, верните массив, содержащий две половины числа. Если число нечетное, увеличьте самое правое число.
Ссылка на задание: https://edabit.com/challenge/xsi99TwpGyFC8KS6d

**Выходные данные:** [2, 2]</br>
